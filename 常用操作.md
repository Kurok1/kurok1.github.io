## mysql新增库和用户，并进行绑定
```sql
CREATE DATABASE new_database CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
CREATE USER 'new_user'@'%' IDENTIFIED BY 'password123'; 
GRANT ALL PRIVILEGES ON new_database.* TO 'new_user'@'%'; 
FLUSH PRIVILEGES; 
EXIT;
```

## PostgreSQL新增库和用户，并进行绑定
```sql
-- 1. 创建数据库 
CREATE DATABASE test_db; 
-- 2. 创建用户 
CREATE USER test_user WITH PASSWORD 'password123'; 
-- 3. 授予用户对数据库的权限 
GRANT ALL PRIVILEGES ON DATABASE test_db TO test_user; 
-- 4. 将数据库所有权授予用户（可选） 
ALTER DATABASE test_db OWNER TO test_user;
```
